✓ Confirmed Decisions
  • ✅
    Same Page Structure — Rebuild preserves the existing page structure: Home, Custom Rods (with sub-pages), In-Stock Rods (with sub-pages), Who We Are, Feedback, Get in Touch, Privacy Policy. No pages being removed or added.
  • ✅
    Shopify for Commerce — Shopify replaces Square as the commerce backend. Handles payment processing, order management, inventory, shipping labels, tax, fraud, and customer data — all in one dashboard. Customer never sees Shopify — they see the custom Next.js frontend.
  • ✅
    Dual Checkout Flows — Standard products (in-stock rods, reels, apparel) use Shopify Storefront API cart → hosted checkout. Custom rod builder uses Admin API → draft orders → hosted checkout. Both flows land in the same Shopify dashboard.
  • ✅
    Draft Orders for Rod Builder — Custom rod configurations are submitted as structured draft orders via Shopify Admin API. Server-side price calculation, validated selections, every option is a structured field — not a note. This was the key architectural decision over line item properties.
  • ✅
    Sanity CMS Continues — All page content, copy, marketing banners, photography, and non-commerce content stays in Sanity. Client manages content without calling Jordan.
  • ✅
    Custom Rod Builder Flow Preserved — The multi-step configurator (choose type → choose series → select application/options → running total) works and should be preserved in the rebuild. UI/UX improvements welcome, logic stays the same. Validation ensures only valid combinations are shown.
  • ✅
    Same Domain — candjcustomrods.com. No domain changes. Jordan handles any DNS or hosting adjustments at launch.
  • ✅
    Next.js + Tailwind + Vercel Stack — Rebuild stays on Next.js 13+ (App Router), React, Tailwind CSS, Framer Motion, Vercel hosting.
  • ✅
    Client Monthly Cost: $39/mo — Shopify Basic plan. Vercel and Sanity remain on free tiers. Down from the fragmented Square + third-party shipping costs of the previous setup.
  • ✅
    Jordan Writes Copy — Final copy refinements for the rebuild come from Jordan. Client provides bullet points and direction, Jordan crafts the final versions.
  • ✅
    Footer Structure — Footer includes: nav links (Custom Rods, Ice Rods, Open Water Rods, Who We Are, Feedback, Get in Touch), social media links, logo, Privacy Policy link, copyright, and "Powered by LatzWebDesign" credit.
  • ✅
    Mobile-First — The rebuild is mobile-first. Nav dropdowns, product grids, rod builder steps, and all content must work cleanly on phones. The current site is already responsive — the rebuild improves it.
