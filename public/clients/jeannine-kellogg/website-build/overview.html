Itasca Books API — Technical Details

What Itasca Is Book distributor / fulfillment service. Division of Bookmobile. Formed 2004. Serves independent publishers.
How It Works Publisher's ecommerce site pushes electronic orders to Itasca. Itasca picks, packs, ships. Publisher handles payment collection + tax.
Backend System iPub distribution system with custom API built by iPro Business Systems. "eFirst" order hub processes orders from any electronic source.
Complexity Warning Three systems communicate in real-time: order hub, distribution system, printshop. Itasca's own integration "was not without its bumps" and took 2.5 years. Jeannine is aware — "I think we need to be prepared for hiccups."
Jeannine's Itasca Account Uses "Stacks" system — can see inventory, initiate drop ships. Has existing relationship.
API Contact Hans (Jeannine's contact). Also: Mark Jung (general distribution, per Itasca website).
Key Quote from Jeannine "I want you doing the development, not them. Hopefully, they can just hand us the 'specs' for the integration."
